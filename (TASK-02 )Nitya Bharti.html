S.NO. HTML NAMES DETAILS HTML TAGS
(i) Table tag This tag is used to display data in a tabular form. <table>
(ii) Table row tag It defines row in a table. <tr>
(iii) Table header tag It defines a table header. <th>
(iv) Table data tag It represents a data cell in a table. <td>
(v) Break tag It is uded to create a line break in a text. <br>
(vi) Span tag This tag is used for the grouping of inline elements. <span>
(vii) Division tag It defines a division or section within HTML document. <div>
